## Getting Started

To get a local copy up and running follow these simple example steps.


### Prerequisites

- Ruby v2.7.2
- Ruby on Rails v6.1.3.1
- Postgres: >=9.5
- Rspec
- Heroku

### Setup

Instal gems with:

- `bundle install`


Setup database with:
 - `rails db:create`
 - `rails db:migrate`


### Usage

Start server with:

- `rails s`


Open `http://localhost:3000/` in your browser.

### Run tests

- `rspec`
